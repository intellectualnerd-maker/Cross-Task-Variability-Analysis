# Neurosious: Cross-task Handwriting Variability for Early Alzheimer’s Detection

## Project Overview
Neurosious is a Python-based research project focused on detecting early signs of Alzheimer’s disease through analysis of handwriting patterns. The project uses machine learning techniques to measure cross-task variability in handwriting features, aiming to identify cognitive instability in individuals.
